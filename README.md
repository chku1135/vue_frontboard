# 프론트엔드 (Vue.js) 프로젝트

본 프로젝트는 Spring Boot 백엔드와 연동되는 Vue.js 기반의 관리자 웹 애플리케이션입니다.

## 주요 기술 스택
- Vue.js 3
- Vite
- TypeScript

## 시작하기
1. Node.js (LTS 버전 권장)가 설치되어 있어야 합니다.
2. 의존성 설치: `npm install`
3. 개발 서버 실행: `npm run dev`
4. 프로덕션 빌드: `npm run build`

## 프로젝트 구조
- `src/`: 소스 코드
- `public/`: 정적 리소스
- `dist/`: 빌드 결과물
